# RpicAcg
一个使用react-native做的第三方哔咔漫画客户端，支持自动打卡。

## 关于 App
练习项目。RN初学者水平，也没有原生开发经验，还在摸索当中，代码主要讲究一个能跑就行。欢迎指导或者是提出Issues。

## 功能
基本功能已经实现，接下来计划增加多种漫画阅读方式（目前只支持条漫模式）以及增加下载功能与账号编辑功能。
